Abaixo temos um exemplo de iframe que tenta capturar alguns dados do usuário.

Utilizando o atributo sandbox com valor sandbox, criamos uma segurança um pouco maior porque não permite que nada seja enviado. Podemos utilizar também sandbox = "allow-same-origin allow-forms allow-cripts" para permitir formulários e scripts da mesma origem.

A referrerpolicy padrão é no-referrer-when-downgrade. Para limitar mais e tornar um pouco mais seguro, colocamos no-referrer (não coleta nenhum tipo de dado do usuário).
